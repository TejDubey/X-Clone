About X Clone

Created by Tej Dubey

This project is a frontend recreation of the X (formerly Twitter) desktop interface. It was built to strengthen my understanding of HTML, CSS, and JavaScript by replicating a real-world social media application with attention to layout, styling, and user interaction.

Project Overview

The goal of this project was not simply to copy the appearance of X, but to understand how modern web interfaces are structured and how interactive components behave.

Features

  • Responsive three-column desktop layout.
  • Sticky navigation and header.
  • Scrollable feed with images and videos.
  • Hover animations and interactive buttons.
  • Floating action buttons.
  • Dropdown "More" menu.
  • Multiple informational pages (Terms, Privacy, Cookies, Accessibility, Ad Information).
  • Modern UI inspired by X.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (Vanilla JS)
  • SVG Icons

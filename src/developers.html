Developer Information

X Clone Project

This project was developed as a frontend recreation of the X (formerly Twitter) desktop interface. It focuses on recreating the visual design, layout, and user interactions using modern web technologies.

Developer

Tej Dubey
Computer Science Student
VIT Chennai

Technologies Used

HTML5
CSS3
JavaScript
SVG Icons

Implemented Features

  • Responsive three-column layout.
  • Sticky navigation sidebar.
  • Fixed top navigation bar.
  • Interactive hover effects.
  • Dropdown "More" menu.
  • Floating action buttons.
  • Embedded images and videos.
  • Additional informational pages.
  • Git & GitHub version control.

Upcoming Features

  • Backend integration.
  • User authentication.
  • Create and delete posts.
  • Like and repost functionality.
  • Comment system.
  • Search functionality.
  • Responsive mobile design.
  • Dark/Light theme switching.

Project Goal

The primary objective of this project is to understand how modern social media interfaces are built while improving frontend development skills through practical implementation.

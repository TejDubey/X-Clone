# X Clone (Frontend)

A frontend recreation of the X (formerly Twitter) homepage built using HTML, CSS, and JavaScript. This project was created to practice frontend development, responsive layouts, DOM manipulation, and UI design by recreating the look and feel of the X web interface.

> **Note:** This project is for educational and portfolio purposes only and is not affiliated with or endorsed by X Corp.

---

## Features

- Three-column X-inspired layout
- Fixed left navigation sidebar
- Fixed right sidebar
- Search bar
- "For You" and "Following" navigation tabs
- Interactive hover effects
- Settings popup
- More menu dropdown
- Premium page
- About page
- Developers page
- Get App page
- Terms of Service page
- Privacy Policy page
- Cookies Policy page
- Accessibility page
- Ads Information page
- External GitHub redirect with confirmation dialog
- SVG icons
- Images and video posts
- Smooth transitions and animations

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Installation

Clone the repository

```bash
git clone https://github.com/TejDubey/X-Clone.git
```

Open the project folder

```bash
cd X-Clone
```

Open `index.html` in your browser.
